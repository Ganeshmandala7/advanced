Help & Documentation

What is FEXsics?

FEXsics (Forensic Expert System for Image Credibility & Authenticity) is a physics-grounded forensic analysis system that detects image manipulation, deepfakes, and AI-generated content using 7 independent forensic engines.

How Does It Work?

The system analyzes your image using multiple forensic techniques:

  • ELA Engine: Detects JPEG compression inconsistencies
  • Metadata Engine: Analyzes camera metadata and timestamps
  • Noise Engine: Checks sensor fingerprints and noise patterns
  • Illumination Engine: Analyzes light source consistency
  • Geometry Engine: Checks perspective and vanishing points
  • Deepfake Engine: Detects GAN-synthesized faces
  • AI-Gen Engine: Detects AI-generated images

Results are combined using Bayesian inference for a final verdict.

Supported Formats

Supported image formats:

  • JPEG (.jpg, .jpeg)
  • PNG (.png)
  • TIFF (.tif, .tiff)
  • WebP (.webp)
  • BMP (.bmp)

Maximum file size: 50MB

Recommended minimum: 32x32 pixels

About Verdicts

Authentic: Image appears to be an unmodified camera capture

Manipulated: Evidence indicates the image has been edited or spliced

Inconclusive: Forensic evidence is mixed; further analysis needed

Privacy & Security

All images are analyzed locally. Chain of custody is maintained via SHA-256 hashing. Analysis results include audit trails suitable for legal proceedings.
